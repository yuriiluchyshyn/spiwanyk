# Парсинг пісень з pryvatri.de

## Опис
Скрипт автоматично парсить пісні з сайту pryvatri.de та додає їх до бази даних MongoDB.

## Можливості
- Парсить пісні з розділів: avtorski, plastovi, povstanski, narodni
- Витягує назву, текст пісні, акорди, YouTube посилання
- Автоматично визначає категорію пісні
- Уникає дублікатів
- Поважає сервер (затримки між запитами)

## Використання

### Запуск парсингу
```bash
# Через npm
npm run scrape

# Через Makefile
make scrape

# Прямо
node src/scripts/scrapeSongs.js
```

### Тестування одної пісні
```bash
npm run scrape-test
```

## Структура парсера

### Розділи сайту
- `avtorski` → категорія `author` (Авторські пісні)
- `plastovi` → категорія `plast` (Пластові пісні)  
- `povstanski` → категорія `uprising` (Повстанські пісні)
- `narodni` → категорія `folk` (Народні пісні)

### Що витягується
1. **Назва пісні** - з посилання на сторінку
2. **Текст пісні** - з основного параграфа сторінки
3. **Акорди** - якщо знайдені в тексті
4. **YouTube посилання** - з iframe або посилань
5. **Автор** - з назви або контенту (якщо доступно)

## Налаштування

### Затримки
- 2 секунди між запитами до різних пісень
- 500мс між запитами до вкладок однієї пісні

### Ліміти
- 20 пісень на розділ (можна змінити в коді)
- Таймаут 15 секунд на запит

## Файли
- `scrapeSongs.js` - основний скрипт парсингу
- `testParser.js` - тест парсингу однієї пісні
- `testSingleSong.js` - детальний аналіз структури сторінки

## Помилки та рішення

### YouTube URL валідація
Скрипт автоматично конвертує відносні YouTube URL в абсолютні.

### Відсутні розділи
Деякі розділи можуть не існувати на сайті - скрипт пропускає їх.

### Дублікати
Перевіряється існування пісні за назвою перед додаванням.
